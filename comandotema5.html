COMANDOS DE TELNET



CONFIGURACION EN EL SERVIDOR

DESPLEGAR

Instalamos el siguiente paquete

sudo apt install openbsd-inetd

Instalamos Telnet

sudo apt install telnetd

Verificat el servidor inetd funciona correctamente

accedemos al siguiente fichero

sudo nano /etc/inetd.conf

verificamos que la linea de telnet no tenga delante el simbolo de #

Reiniciamos el fichero de inetd

sudo /etc/init.d/openbsd-inetd restart

Verificar que el servicio de telenet esta ala escucha

netstat -ltp
netstat -ltpn

Ahora solo nos dirijimos al equipo cliente y escribimos el siguiente comando

Telnet direccion ip / Nombre del equipo >

VERIFICAR SI UN PUERTO ESTA ABIERTO CON TELNET

telnet 22

ACCEDER DESDE UN CLIENTE TELNET (CON EL NOMBRE HOST Y HOSTNAME)

telnet

en este caso estamos entrando con el nombre de la maquina no con la ip

CAMBIAR EL PUERTO DEL SERVIDOR TELNET

sudo nano /etc/services

aqui cambiamos la lena de telnet donde se especifica el puerto

REINICIAR EL SERVIDOR TELNET

sudo /etc/init.d/openbsd-inetd restart

INSTALAR EL SERVIDOR FTP

sudo apt install vsftpd

INICIAR EL SERVIDOR FTP

sudo systemctl status vsftpd.service
como acceder al servidor ftp desde cliente

no hace falta instalar nada en el equipo cliente solo nos pedira que escribimos nuestro nombre de usuario ftp,una ves escrito nuestro nombre nos pedira nuestra contrasena
