COMANDOS DE TELNET



CONFIGURACION EN EL SERVIDOR

DESPLEGAR
Instalamos el siguiente paquete
sudo apt install openbsd-inetd
Instalamos Telnet
sudo apt install telnetd

Verificat el servidor inetd funciona correctamente

accedemos al siguiente fichero

sudo nano /etc/inetd.conf

verificamos que la linea de telnet no tenga delante el simbolo de #

Reiniciamos el fichero de inetd
sudo /etc/init.d/openbsd-inetd restart
Verificar que el servicio de telenet esta ala escucha
netstat -ltp netstat -ltpn
Ahora solo nos dirijimos al equipo cliente y escribimos el siguiente comando
Telnet direccion ip / Nombre del equipo >
